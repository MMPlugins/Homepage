# Homepage plugin for [Dragory's ModMail](https://github.com/dragory/modmailbot)

### Setup: ###
Make sure your running at least v3.0.3 of Modmail.
in your config.ini file, make a new line and add:  
```ini
plugins[] = npm:MMPlugins/Homepage
```
Restart your bot!
And that's it!

Use the [prefix]homepage command and see it for yourself!
